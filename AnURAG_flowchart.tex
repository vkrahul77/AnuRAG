%  ABSTRACT (for paper — compile separately or paste into main .tex)
% ════════════════════════════════════════════════════════════
%
% \begin{abstract}
% The automation of analog integrated circuit (IC) design remains
% fundamentally constrained by the high dimensionality of continuous
% design spaces and the need for rigorous physical intuition.
% Simulation-centric ``black-box'' strategies such as Reinforcement
% Learning and Bayesian Optimization attempt to bridge this gap, yet
% they operate as opaque sizing oracles: each candidate requires a
% full SPICE evaluation, leading to extreme sample inefficiency and
% yielding uninterpretable sizing decisions.  In this work, we
% introduce \textbf{AnuRAG} (\textbf{An}alog Design Framework with
% \textbf{u}nified \textbf{R}etrieval-\textbf{A}ugmented
% \textbf{G}eneration), a two-stage ``white-box'' assistant that
% augments the human designer rather than replacing them.
%
% AnuRAG operates in two explicit stages.  In \emph{Stage~1
% (Topology Selection)}, a ReAct-style agent backed by Gemini~2.0
% performs hybrid retrieval (semantic search over 768-dimensional
% \texttt{text-embedding-004} vectors combined with BM25 keyword
% matching) against a curated knowledge base of 406 sources---402
% IEEE JSSC papers spanning 2000--2024 and four canonical
% textbooks---comprising 17\,769 embedded chunks derived from
% 10\,929 text segments and 16\,132 images.  The agent applies
% physics-grounded feasibility checks (voltage headroom, swing
% constraints, intrinsic-gain limits) to prune infeasible candidates
% and returns a ranked shortlist with explicit literature citations.
% In \emph{Stage~2 (Design-Space Exploration)}, the same RAG engine
% switches to equation-driven code generation: it emits executable
% Python scripts that sweep $g_m/I_D$ look-up tables across the
% design space, producing Pareto frontiers over $>$10\,000 sizing
% points---\emph{without a single SPICE call}.  A sparse
% verification loop then samples $\sim$5 anchor points along the
% frontier for SPICE simulation; if the analytical--simulated error
% exceeds 10\%, the framework automatically re-calibrates its
% equations and regenerates the frontier.
%
% We evaluate AnuRAG on OTA design tasks in the IHP SG13G2 130\,nm
% BiCMOS open-source PDK.  On a folded-cascode benchmark
% (Gain\,$\geq$\,70\,dB, GBW\,$\geq$\,500\,MHz, $C_L{=}500$\,fF),
% Stage~1 returns a ranked topology shortlist from 406 sources in
% under 60\,s.  Stage~2 produces a 10\,000-point Pareto frontier
% whose analytical predictions match SPICE within 8--12\,\% at five
% anchor points, requiring only 5 simulations versus $>$500 for a
% comparable Bayesian-optimization baseline.
% \end{abstract}
%
% ════════════════════════════════════════════════════════════
